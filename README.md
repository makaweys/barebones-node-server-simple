# Simple Node.js Core Server

A minimal HTTP server built with only Node.js core modules - no external dependencies!

## Features

- HTTP server using `http` module
- JSON request parsing
- CORS support
- Static file serving
- Simple routing
- Logging middleware

## Usage

```bash
# Clone and run
git clone git@github.com:makaweys/barebones-node-server-simple.git
node server.js

# Server runs on http://localhost:3000
```
